# the two commands you keep typing:

which one, when

awake session
lid open
awake until the timer ends.
closing the lid sleeps it.
caffeinate
stay awake, lid closed
lid shut
never sleeps, even shut.
screen still sleeps and locks after inactivity.
disablesleep
both
lid might close
awake until you turn it off.
closing the lid changes nothing.
caffeinate + disablesleep
idle screen on screen off, Mac on
